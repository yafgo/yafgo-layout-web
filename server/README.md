## 开始使用

```shell
> ./ycli
[Yafgo-Cli] v1.0.0

Usage:
 ./ycli [command]

Available Commands:
  make      代码生成
  play      代码演练
  orm       生成gorm代码
  migrate   执行db迁移
  doc       更新swagger文档
  serve     启动webServer
```
